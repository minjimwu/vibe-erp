# Vibe ERP 系統操作手冊

歡迎使用 Vibe ERP 進銷存系統。本系統是一款基於瀏覽器的輕量化 ERP，所有資料均儲存在您的電腦本機（IndexedDB），確保資料隱私與極速反應。

---

## 🚀 快速開始順序

為了確保系統運作正確，建議您按照以下順序進行建檔與作業：

1. **客戶資料建檔**：建立您的客戶名單。
2. **廠商資料建檔**：建立您的供應商名單。
3. **產品基本資料與分類**：建立商品品項，並定義分類。
4. **進貨入庫作業**：記錄進貨明細，系統會自動增加庫存。
5. **銷貨出庫作業**：記錄銷售明細，系統會自動扣除庫存。
6. **歷史查詢與報表**：查看銷售狀況與獲利。

---

## 1. 基礎資料建檔

### 客戶資料 (Customers)
*   **進入路徑**：側邊欄 -> 「批次編輯」 -> 選擇「客戶資料」
*   **操作方式**：
    *   本系統採用 **批次編輯 (Spreadsheet)** 模式作為唯一編輯方式。
    *   直接在表格中輸入資料，或從 Excel 複製貼上。
    *   **必填欄位**：客戶編號、客戶名稱。
*   **用途**：在進行銷貨作業時，需先有客戶資料才能建立訂單。

### 廠商資料 (Suppliers)
*   **進入路徑**：側邊欄 -> 「批次編輯」 -> 選擇「進貨廠商」
*   **操作方式**：
    *   同樣採用 **批次編輯** 模式。
    *   **必填欄位**：廠商編號、廠商名稱。
*   **用途**：在記錄進貨時，關聯進貨來源。

### 產品分類與商品建檔 (Products)
*   **進入路徑**：側邊欄 -> 「批次編輯」 -> 選擇「商品庫存」
*   **操作方式**：
    *   請在批次編輯表格中建立商品代碼、名稱與類別。
    *   **產品類別**：在此欄位輸入類別名稱（如：電子產品、文具等），銷貨時可依此類別篩選。
    *   **價格設定**：
        *   **進貨成本**：商品的基本進貨價。
        *   **預計售價**：您預計賣給客戶的金額。
*   **小技巧**：建議先在「批次編輯」模式中，將所有產品類別與代碼整理好後一次匯入。

---

## 2. 進貨管理 (Purchases)

### 進貨 Key-in
*   **進入路徑**：側邊欄 -> 「進貨管理」 -> 「新增進貨」
*   **作業內容**：
    1.  確認**進貨單號**與**日期**。
    2.  **選擇廠商**：從下拉選單選擇供應商。
    3.  **選擇產品**：從下拉選單選擇產品，系統會自動帶入「進貨成本」。
    4.  **數量**：輸入進貨數量。
*   **庫存連動**：儲存後，系統會自動將該商品的「庫存量」增加對應的數量。

---

## 3. 庫存調整 (Inventory Adjustment)

如果您需要校正庫存數量（例如：盤損、贈送）：
*   **操作方式**：進入側邊欄 -> 「批次編輯」，選擇「商品庫存」，直接修改「庫存量」欄位後點擊「提交更新」。

---

## 4. 銷貨作業 (Sales)

### 銷貨流程
*   **進入路徑**：側邊欄 -> 「出貨管理」 -> 「新增出貨」
*   **操作步驟**：
    1.  **選客戶**：從下拉選單中選擇客戶名稱。
    2.  **選類別與產品**：先選擇「產品類別」，再從篩選後的「產品項目」中選擇商品。
    3.  **價格確認**：系統會自動帶入該產品的「預計售價」，您仍可手動調整單價。
    4.  **數量輸入**：輸入本次銷售數量（預設為 1）。
    5.  **總計金額**：系統會根據 `單價 x 數量` 自動計算總金額。
*   **庫存連動**：儲存後，系統會自動從「庫存量」扣除該數量。

---

## 5. 查詢與報表 (Reports)

### 歷史查詢
*   **所有紀錄**：可在「進貨管理」與「出貨管理」分頁查看完整的流水帳。
*   **分類查詢**：在「銷售報表」分頁中，您可以透過下拉選單：
    *   **選客戶**：篩選特定客戶的所有購買歷史。
    *   **查看總額**：系統會自動統計該篩選範圍內的「查詢總營業額」。

---

## 💡 進階功能：批次編輯 (Spreadsheet)

如果您有大量資料要從 Excel 搬過來：
1.  進入「批次編輯」。
2.  選擇要編輯的表格（商品、廠商或客戶）。
3.  直接從 Excel 複製資料列，在系統表格中貼上 (Ctrl+V)。
4.  點擊「提交更新」完成大量建檔。

---

## ⚠️ 重要維護須知

*   **資料備份**：由於資料存在瀏覽器本機，請定期點擊右上角的**「匯出本機資料」**下載 Excel 備份。
*   **資料還原**：更換電腦或清除瀏覽器緩存後，可點擊**「匯入 Excel」**將備份檔恢復。
*   **清除資料**：若需重整系統，可在「系統資料」分頁執行「清除所有本機資料」。
