# Vibe ERP - 純前端進銷存系統

這是一套設計優雅、反應流暢的「純前端」進銷存管理系統。專為需要快速、安全且無須架設後端的個人工作室或微型企業設計。

## 🌟 核心特點

- **100% 純前端運行**：所有資料均儲存在您的瀏覽器本機資料庫 (IndexedDB) 中，無需伺服器，資料絕對隱私。
- **高質感介面**：採用 Glassmorphism (玻璃擬物) 設計語法，提供極佳的視覺體驗與流暢的動態效果。
- **強大的 Excel 整合**：
  - **範本下載**：一鍵產生標準格式的 Excel 範本。
  - **批次匯入**：支援匯入舊有的商品與廠商資料。
  - **資料備份**：隨時可以將本機資料庫的所有內容匯出為 Excel 檔案備份，或轉移至其他電腦。
- **離線可用**：只要有瀏覽器，即使沒有網路也能正常作業。

## 🛠️ 主要功能

1. **總覽看板 (Dashboard)**：即時視覺化銷售趨勢圖與庫存預警。
2. **商品管理**：支援產品代碼、類別、進成本、預計售價及庫存自動增減。
3. **進貨/出貨管理**：完整紀錄每一筆交易明細，自動同步更新庫存量。
4. **廠商/客戶管理**：集中管理合作夥伴與客戶聯絡資料。
5. **銷售報表**：可根據客戶篩選出貨紀錄，並自動加總營業額。

## 🚀 如何開始

1. 克隆 (Clone) 本專案到本機。
2. 直接使用瀏覽器開啟 `index.html`。
3. 在頁面右上角點擊「下載 Excel 範本」。
4. 根據範本填寫資料後，點擊「匯入 Excel」即可開始使用。

## 📋 注意事項

- **定期備份**：由於資料儲存在瀏覽器快取中，清空瀏覽器快取可能會導致資料遺失。請務必定期使用內建的「匯出」功能進行 Excel 備份。
- **資料轉移**：若要更換電腦，請將在舊電腦匯出的 Excel 檔案在新電腦中再次「匯入」即可無縫接軌。
